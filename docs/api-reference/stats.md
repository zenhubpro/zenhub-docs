# Stats

Dashboard analytics e metricas de campanhas.

## Dashboard da organizacao

<span class="api-method api-method--get">GET</span> `/v1/stats/dashboard`

Retorna overview geral: campanhas ativas, total de grupos, membros, agendamentos e mensagens.

---

## Estatisticas de campanha

<span class="api-method api-method--get">GET</span> `/v1/stats/campaigns/:id`

Retorna metricas detalhadas de uma campanha especifica.
