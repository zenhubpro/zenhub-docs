# Conector Claude.ai

:::info Em desenvolvimento
O conector remoto via HTTP/SSE com OAuth 2.1 + PKCE em `https://mcp.zenhub.pro` esta em desenvolvimento. Use o transport stdio com Claude Desktop ou Claude Code enquanto isso.
:::

Quando disponivel:

1. Acesse **claude.ai > Settings > Connectors**
2. Clique em **"Add custom connector"**
3. URL: `https://mcp.zenhub.pro/mcp`
4. Faca login com sua conta ZenHub
5. Selecione a organizacao
6. Pronto — as ferramentas aparecem em qualquer chat

A diferenca para o stdio e que voce nao precisa instalar nada localmente nem gerenciar API keys — o OAuth amarra o conector a sua conta ZenHub.
