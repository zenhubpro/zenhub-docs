# ZenHub

Bem-vindo a documentacao do **ZenHub** — plataforma de automacao de comunicacao WhatsApp.

O ZenHub oferece tres formas de integracao:

| Metodo | Ideal para | Setup |
|--------|-----------|-------|
| **API REST** | Sistemas externos, automacoes, bots | API Key via dashboard |
| **CLI** | Terminal, scripts, CI/CD | `npm install -g zenhub-cli` |
| **MCP** | IAs (Claude Desktop, Codex, Cursor, Copilot) | `npx zenhub-mcp` |

## O que voce pode fazer

- **Campanhas** — Criar e gerenciar campanhas WhatsApp com grupos automaticos
- **Agendamentos** — Programar mensagens para datas e horarios especificos
- **Mensagens** — Enviar texto, imagens, videos, documentos e enquetes
- **Grupos** — Gerenciar grupos, membros, admins e configuracoes
- **ZenChat** — Inbox omnichannel: listar conversas, enviar respostas, gerenciar contatos
- **Lista de Acesso** — Controle de whitelist por campanha com expiracao
- **Blacklist** — Bloquear numeros indesejados
- **Webhooks** — Receber notificacoes em tempo real de eventos
